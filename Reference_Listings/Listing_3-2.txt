# Azure Deployment Review Skill

When reviewing an Azure deployment, check the following areas:

1. Confirm the target Azure service and environment.
2. Review identity, secrets, and access assumptions.
3. Check logs, metrics, alerts, and rollback requirements.
4. Identify quota, cost, networking, and security risks.
5. Summarize findings as required actions, recommended
   improvements, and open questions.
