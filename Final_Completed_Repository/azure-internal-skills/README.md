# Azure Internal Skills

This repository stores reusable Azure engineering skills. Edit canonical skills under `skills/`. Install approved copies into a supported host path such as `.github/skills/`.

## First skill to test

1. Open the repository root in Visual Studio Code.
2. Open GitHub Copilot Chat in Agent mode.
3. Confirm `azure-app-service-deployment-review` is visible.
4. Ask it to review `skills/azure-app-service-deployment-review/examples/app-service-design-brief.md`.

Do not place credentials, connection strings, customer information, or production incident data in this repository.
