# Azure Monitor Investigation

Collect the affected resource, environment, UTC time window, symptom, impact, and recent changes. Identify the metrics, logs, traces, activity events, and resource-health evidence required. Separate observations from hypotheses and do not claim that telemetry was queried unless results are available.
