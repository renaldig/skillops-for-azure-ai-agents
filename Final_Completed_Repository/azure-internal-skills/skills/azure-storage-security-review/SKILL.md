# Azure Storage Security Review

This starter file is intentionally brief. Expand it when the skill becomes an approved library candidate.

Before giving a recommendation, ask for the storage service, data sensitivity, network boundary, identity model, public-access settings, shared-key requirements, encryption requirements, diagnostic settings, and retention needs.

Return blockers, recommendations, open questions, and next actions. Do not claim compliance from configuration details alone.
