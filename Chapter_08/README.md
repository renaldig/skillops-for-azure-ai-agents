# Chapter 8 supplementary files

Exercise 8-1 uses the canonical and installed `azure-ai-search-retrieval-design` skill with `retrieval-brief.md`.

Exercise 8-2 uses the three fictional policy files, `policy-retrieval-toolbox.yaml`, the Foundry Search integration record, and the hosted-agent source reference. The hosted-agent reference contains `main.py`, `Dockerfile`, `requirements.txt`, `.env.example`, `toolbox.yaml`, `.dockerignore`, `.azdignore`, and an `azure.yaml` reference.

The Dockerfile is included for transparency and optional container builds. Current Foundry scaffolds can deploy Python code directly from the consolidated `azure.yaml`, so follow the current scaffold when it differs.
