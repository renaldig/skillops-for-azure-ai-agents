# Chapter 3 supplementary files

`skillops-chapter3-lab` is a ready-to-open Visual Studio Code workspace for Exercises 3-1 and 3-2. The skill is already placed under `.github/skills` so the reader can focus on discovery and activation. The `.vscode/mcp.json` file starts Azure MCP Server with a read-only tool surface.

Exercise 3-2 still requires an Azure subscription and an approved non-production resource group. `create-sandbox-resource-group.ps1` is an optional helper for readers who do not already have one.
